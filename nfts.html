NFT Catalog

Q‑BOND Network utilises a suite of non‑fungible tokens to manage identities, incentives and legacy within the protocol. Each NFT category serves a specific role and is minted under defined conditions. Soulbound tokens cannot be transferred, while other NFTs may be tradable or inherited.

QIE Identity NFT

  • Type: Soulbound
  • Trigger: Minted on first stake
  • Purpose: Sovereign ID, DeFi tax ID, memory upload key, messaging handle and identity proof

XP NFT

  • Type: Earnable soulbound
  • Trigger: Milestone‑based validator activity
  • Purpose: Determines yield curve boosts, increases proposal weight, unlocks perks and records trust

Validator Genesis NFT

  • Type: Prestige badge (optional tradable)
  • Trigger: Awarded to top 100 early validators
  • Purpose: Proof‑of‑origin, permanent yield multiplier, legacy badge

Proof‑of‑Research NFT

  • Type: Project‑specific validator‑voted NFT
  • Trigger: Minted after proposal passes DAO vote
  • Purpose: Linked to reproducible IP, yield tied to scientific royalties and contribution history

Memory Resurrection NFT

  • Type: zkProof‑attached NFT
  • Trigger: Validator logs verified historic contributions
  • Purpose: Stores legacy XP, can be inherited, adds weight in Memory Vault DAO

Bloodline Link NFT

  • Type: Succession certificate
  • Trigger: Heir designated through governance
  • Purpose: Grants succession rights and unlocks XP inheritance

Sovereign Signal NFT

  • Type: Multi‑DAO credential
  • Trigger: Earned through cross‑DAO contributions
  • Purpose: Acts as a DAO passport, grants XP boost and enables access to special treasuries

Aurora‑Linked Dynamic NFT

  • Type: AI‑reactive NFT
  • Trigger: Validator signal resonance via Aurora.EXE
  • Purpose: Tracks behaviour, evolves with activity, ties into trust scoring

Validator Referral NFT

  • Type: Gamified badge (tradable)
  • Trigger: Earned after onboarding two or more validators
  • Purpose: Tracks network growth and grants XP or treasury bonuses

Q‑BOND Timelock NFT

  • Type: Yield‑bearing time‑lock
  • Trigger: Issued when staking into a timelock vault
  • Purpose: Defines yield, maturity and rebond terms; tradable pre‑maturity and usable in treasury‑backed pools

The NFT framework is designed to incentivize participation and align values across the ecosystem. Details are based on internal communications.
